# MiroFish Windows Task Scheduler Setup

Automate `mirofish_sender.py` to run every 55 minutes on your Windows PC,
so it fires before the hourly cron on the VPS.

---

## Manual Setup via Task Scheduler GUI

1. **Open Task Scheduler**
   - Press `Win + S`, search for "Task Scheduler", press Enter

2. **Click "Create Basic Task"** in the right-hand Actions panel

3. **Name the task:**
   - Name: `MiroFish Limitless AI`
   - Description: `Runs 250-agent MiroFish sentiment simulation and sends to VPS`

4. **Set the Trigger:**
   - Select: `Daily`
   - Click Next → set start time to current time
   - Click Next
   - **Important:** After finishing the wizard, go back and edit the trigger:
     - Open the task → Triggers tab → Edit trigger
     - Check "Repeat task every: **55 minutes**" for a duration of "Indefinitely"
     - This ensures it fires every 55 minutes all day

5. **Set the Action:**
   - Select: `Start a program`
   - Program/script: Path to your Python executable
     - Find it by running in Command Prompt: `where python`
     - Example: `C:\Python312\python.exe` or `C:\Users\YourName\AppData\Local\Programs\Python\Python312\python.exe`
   - Arguments: `C:\mirofish_sender.py`
   - (Make sure mirofish_sender.py is saved at C:\mirofish_sender.py)

6. **Security Options:**
   - Check: ✅ `Run whether user is logged on or not`
   - Check: ✅ `Run with highest privileges`
   - Configure for: Windows 10/11

7. **Finish** → Enter your Windows password when prompted

## Automatic Setup via PowerShell (Recommended)

Run this in an **elevated PowerShell** (Run as Administrator):

```powershell
$action = New-ScheduledTaskAction -Execute "python" -Argument "C:\mirofish_sender.py"
$trigger = New-ScheduledTaskTrigger -RepetitionInterval (New-TimeSpan -Minutes 55) -Once -At "00:00"
Register-ScheduledTask -TaskName "MiroFish Limitless AI" -Action $action -Trigger $trigger -RunLevel Highest
```

### Verify the task was created:
```powershell
Get-ScheduledTask -TaskName "MiroFish Limitless AI"
```

### Run it manually right now (test):
```powershell
Start-ScheduledTask -TaskName "MiroFish Limitless AI"
```

### Remove the task (if needed):
```powershell
Unregister-ScheduledTask -TaskName "MiroFish Limitless AI" -Confirm:$false
```

## Notes

- **Why 55 minutes?** The VPS runs its trading pipeline on the hour. MiroFish at 55 min ensures fresh sentiment data is always available.
- **mirofish_sender.py location:** The script can be placed anywhere. Update the path in Task Scheduler accordingly.
- **Python path:** Use `where python` in Command Prompt to find the exact path. If using a virtual environment, use the venv's python.exe.
- **Ollama required:** mirofish_sender.py uses Ollama + Mistral 7b locally. Ensure Ollama is running: `ollama serve`
- **VPS fallback:** If your PC is off, the VPS can generate its own simulation via: `curl http://167.71.25.250:9876/simulate`

## Downloading mirofish_sender.py to Your PC

The latest version is always available from the VPS:

```powershell
# Download to C:\
Invoke-WebRequest -Uri "http://167.71.25.250:9876/sender_script" -OutFile "C:\mirofish_sender.py"
```

Or via curl:
```
curl http://167.71.25.250:9876/sender_script -o C:\mirofish_sender.py
```

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Task runs but no output | Add `-OutFile C:\mirofish_log.txt` to capture output |
| Python not found | Use full path in Task Scheduler, e.g. `C:\Python312\python.exe` |
| VPS unreachable | Check VPS is online: `ping 167.71.25.250` |
| Ollama not running | Start it: `ollama serve` (keep it running in background) |
| Task won't repeat | Edit trigger → check "Repeat task every 55 minutes" box |
